# MMM-DNews
Danish news


News feed with picture ...

MagicMirror/modules 

git clone https://github.com/cowboysdude/MMM-DNews

Go to: MMM-Dnews
Type: npm install

Add to Config.js

     {
           module: 'MMM-DNews',
          position: 'bottom_bar',
          config: {
			     rotateInterval: 20 * 1000 //changes every 20 seconds.
			    }
    },
    
    Enjoy!
